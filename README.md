# To-Do-List
Yapılacak olan görevlerinizi, kelime bazlı olarak size göre önceliklerini sıralayan, girdiğiniz tarihe göre yapmanız gerekenleri belirten bir uygulama.

Dosyayı indirdikten sonra live-server ile çalıştırınız. Ardına terminalde gözüken link üzerinden giriş yaptığınız zaman istediğiniz görevleri, işinizi, unutmamanız gereken şeyleri yazabilirsiniz.

Veri tabanı yerel olduğu için bir text dosyanızda bu görevler yazı olarak duracaktır. Tekrardan çalıştırdığınızda görev, iş ve unutmak istemediğiniz şeyler hatırlatıcı ile karşınıza çıkacaktır.
